# @worldwideview/wwv-lib-incidents

Shared abstract base classes for Linje.track incident plugins (earthquakes, wildfires, conflicts)

## Usage
This package is part of the Linje.track plugin ecosystem. It provides the following capabilities:
- **ID:** `wwv-lib-incidents`
- **Category:** Miscellaneous
- **Format:** bundle

## Installation
Typically installed via the Linje.track Marketplace or discovered automatically.
If installing manually in a Next.js setup:
```bash
npm install @worldwideview/wwv-lib-incidents
```

## Architecture
This plugin adheres to the Linje.track standard plugin structure. As a `bundle` plugin, it connects to the core Event Bus and renders map capabilities.
